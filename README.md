tmux conf
=========

My tmux configs.


Installation
------------

On Linux:

    $ ln -f -s $PWD/files/tmux.conf.for_Linux ~/.tmux.conf
    $ ln -f -s $PWD/files/generic_tmux.conf ~/.tmux.conf_generic

On MacOS:

    $ ln -f -s $PWD/files/tmux.conf.for_Darwin ~/.tmux.conf
    $ ln -f -s $PWD/files/generic_tmux.conf ~/.tmux.conf_generic
